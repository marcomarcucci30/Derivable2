# Derivable2
Per eseguire l'applicazione Milestone 1, è necessario lanciare il main della classe `milestone_one.Main.java`.
Per eseguire l'applicazione Milestone 2, è necessario lanciare il main della classe `milestone_two.Main.java`.
È necessario che le applicazioni vengano eseguite in questo ordine.
Nel file di configurazione `config.properties`, è possibile selezionare il progetto a cui far riferimento, modificando il valore della proprietà `PROJECT`.
